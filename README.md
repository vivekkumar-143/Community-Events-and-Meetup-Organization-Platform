# 🌐 Community Events & Meetup Organization Platform

A full-stack web application that allows users to create, manage, and join community events easily. This platform helps people discover local events and connect with others.

---

## 🚀 Features

- 📝 Create and manage events
- 🔍 Browse and explore events
- 📍 Add event details (location, date, time)
- 👥 Join events
- 📊 Event capacity management
- ⚡ Real-time event listing

---

## 🛠️ Technologies Used

### Frontend
- React.js
- HTML, CSS, JavaScript

### Backend
- Python (Flask) / Node.js (based on your setup)
- REST API

### Other Tools
- Git & GitHub
- VS Code
